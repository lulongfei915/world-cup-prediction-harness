# Recommended Sources

使用这个 harness 时，先从可信来源确认事实，再让 Agent 分析。

## 事实来源

- FIFA 官方比赛中心
- ESPN Soccer 赛程和赛果页
- 各国家队或足协官方账号
- 赛事官方社交账号

## 数据来源

- FIFA 官方技术统计
- ESPN 比赛数据
- FBref
- Opta 或 Stats Perform 公开预测
- Elo ratings

## 新闻来源

- 主帅赛前发布会
- 国家队官方伤停公告
- 可信体育媒体的赛前报道
- 球场当地天气预报

## 使用规则

1. 赛程、开球时间、比分必须优先使用官方或权威来源。
2. 新闻类信息必须区分事实、传闻和观点。
3. 同一信息如果被多家媒体引用，仍要检查是否来自同一个原始源。
4. 所有关键结论必须保留链接，方便赛后复盘。
